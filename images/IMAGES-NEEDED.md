## Where to Find Images

You can find appropriate images from:
1. **Official Harry Potter sources** (be mindful of copyright)
2. **Free stock photo sites** like Unsplash, Pexels
3. **AI image generators** like DALL-E, Midjourney, Stable Diffusion
4. **Fan art sites** (with proper attribution)
5. **Create placeholder images** using online tools

## Image Specifications
- **Crests**: 150px height, transparent PNG preferred
- **Faculty/Students**: 300x400px recommended
- **Founders**: 200px width for sidebar display
- **File format**: JPG or PNG
- **Optimization**: Compress images for web use

## Quick Setup
1. Create an `images` folder in your project root if it doesn't exist
2. Download or create the images
3. Name them exactly as listed above
4. Place them in the `images/` folder
